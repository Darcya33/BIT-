角色主页说明

登录后系统将根据身份加载对应功能入口和任务说明。

  • 管理员可查看系统概览并维护企业基础信息。

审核工作说明

  • 企业当前可先保存就业和失业草稿，再逐条提交到市级审核队列。
  • 市级审核员已可以看到待审记录，并执行通过或退回动作。
  • 省级审核员已可以处理市级通过后的待审记录，并形成终审通过或省级退回。
